## How to update the existing GitHub site

1. Unzip this folder
2. Open your GitHub repository
3. Upload/replace:
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`
4. Commit changes
5. Wait a minute or two for GitHub Pages to refresh
